Kurze Definitionen zentraler Begriffe der Zauberkunst. Für ausführlichere Erklärungen siehe die verlinkten Seiten.

## A

**Angle** (auch: *Winkel*)
: Die Sichtlinie des Zuschauers. Manche Techniken funktionieren nur aus bestimmten Winkeln — „angle-proof" bedeutet, dass der Trick von allen Seiten funktioniert.

## B

**Break**
: Eine unsichtbare Lücke im Kartendeck, gehalten vom kleinen Finger oder Daumen. Markiert eine Position im Deck. → [Break](/techniken/controls/break)

**Burnout**
: Wenn ein Zuschauer eine Hand oder ein Objekt zu intensiv beobachtet. „Er hat meine Hand geburned" — der Zuschauer hat genau hingeschaut, wo er nicht hinschauen sollte.

## C

**Clean** (auch: *sauber*)
: Ein Zustand, in dem alles geprüft werden kann — keine versteckten Objekte, keine Gimmicks in der Hand. „Am Ende bist du clean" bedeutet, dass nach dem Trick nichts Verdächtiges zurückbleibt.

**Close-up**
: Zauberkunst aus nächster Nähe, oft am Tisch. → [Close-up](/sparten/close-up)

**Control**
: Eine Technik, um eine Karte unbemerkt an eine bestimmte Position im Deck zu bringen. → [Controls](/techniken/controls/)

## D

**Dirty**
: Gegenteil von *clean* — der Magier hat ein verstecktes Objekt in der Hand oder ein Gimmick ist noch im Spiel.

**Ditch**
: Ein verstecktes Objekt unauffällig loswerden — z.B. ein Gimmick in die Tasche fallen lassen.

**Double Lift**
: Zwei Karten als eine vom Deck abheben und zeigen. → [Double Lift](/techniken/double-lift)

## E

**Effekt**
: Was der Zuschauer wahrnimmt — das Ergebnis des Tricks aus Zuschauerperspektive. „Der Effekt ist ein Verschwinden" bedeutet: der Zuschauer sieht, dass etwas verschwindet. → [Effekte](/effekte/)

## F

**False Shuffle** (dt.: *Falschmischen*)
: Scheinbar mischen, tatsächlich die Kartenreihenfolge beibehalten. → [Falschmischen](/techniken/falschmischen)

**Flourish**
: Eine sichtbare Demonstration von Geschicklichkeit — nicht täuschend, sondern beeindruckend. Z.B. [Card Spring](/techniken/praesentation/card-spring), [Coin Roll](/techniken/muenzen/coin-roll), [Fanning](/techniken/praesentation/fanning).

**Force**
: Den Zuschauer scheinbar frei wählen lassen, tatsächlich aber die Wahl bestimmen. → [Forces](/techniken/forces/)

**French Drop**
: Klassische falsche Münzübergabe. → [French Drop](/techniken/muenzen/french-drop)

## G

**Gaff Card** (auch: *Gimmicked Card*)
: Eine präparierte Karte — z.B. doppelseitig bedruckt, mit anderem Rücken, oder mit einem Spezialeffekt versehen.

**Gimmick**
: Ein geheimes Hilfsmittel, das der Zuschauer nicht sehen darf. Kann ein präpariertes Objekt, eine versteckte Vorrichtung oder ein spezielles Requisit sein. → [Gimmick](/konzepte/gimmick)

## H

**Handling**
: Die spezifische Art, wie ein Trick technisch ausgeführt wird. Ein Trick kann verschiedene Handlings haben — gleiches Ergebnis, andere Methode.

## I

**Impromptu**
: Ein Trick, der ohne Vorbereitung mit geliehenen Gegenständen funktioniert. Das Gegenteil von einem Trick, der ein Gimmick oder Setup benötigt.

## K

**Key Card**
: Eine Karte, deren Position der Magier kennt — dient als Referenzpunkt, um eine andere Karte zu finden.

## L

**Lapping**
: Objekte unauffällig in den Schoß fallen lassen, während man am Tisch sitzt. Eine klassische Ditch-Methode.

## M

**Methode**
: Die geheime Funktionsweise eines Tricks — die Technik, das Gimmick oder das Prinzip hinter dem Effekt.

**Misdirection**
: Die Kunst, die Aufmerksamkeit des Zuschauers zu lenken — weg von der Methode, hin zum Effekt. → [Misdirection](/konzepte/misdirection)

## O

**Out**
: Ein Ausweg, wenn etwas schiefgeht. „Ich habe einen Out" — es gibt einen Plan B, falls der Trick nicht wie geplant funktioniert.

## P

**Palmieren** (engl.: *Palming*)
: Ein Objekt unsichtbar in der Hand verbergen. → [Palmieren](/techniken/palmieren)

**Patter**
: Der gesprochene Text während einer Vorführung — Geschichten, Erklärungen, Witze. → [Patter](/techniken/praesentation/patter)

**Peek**
: Heimlich auf eine Karte schauen, die der Magier nicht kennen sollte.

**Produktion**
: Ein Objekt aus dem Nichts erscheinen lassen.

## R

**Reset**
: Den Trick zurück in den Ausgangszustand bringen, damit er sofort wiederholt werden kann.

**Routine**
: Eine zusammenhängende Abfolge mehrerer Tricks oder Phasen, die eine Geschichte erzählen. → [Routine](/konzepte/routine)

## S

**Self-working** (dt.: *selbstarbeitend*)
: Ein Trick, der ohne Sleight of Hand funktioniert — basierend auf mathematischen Prinzipien, Gimmicks oder geschicktem Aufbau.

**Setup**
: Die geheime Vorbereitung vor einem Trick — z.B. Karten in einer bestimmten Reihenfolge anordnen.

**Sleight** (auch: *Sleight of Hand*, *Move*)
: Eine unsichtbare manuelle Technik. → [Sleight of Hand](/techniken/)

**Steal**
: Ein Objekt heimlich in die Hand nehmen, ohne dass der Zuschauer es bemerkt — das Gegenteil von *ditch*.

## T

**Thumb Tip** (dt.: *Daumenspitze*)
: Ein fleischfarbener Hohlkörper, der über den Daumen gestülpt wird. Ermöglicht das Verschwinden kleiner Objekte (Tücher, Papier). Eines der vielseitigsten Gimmicks. → [Thumb Tip](/requisiten/thumb-tip)

**Top Change**
: Die oberste Karte des Decks wird unsichtbar gegen eine andere Karte ausgetauscht.

**Trick**
: Die gesamte Vorführung — Effekt, Methode und Präsentation zusammen.
