Falschmischen (engl. *False Shuffle*, franz.: *faux mélange*) ist eine Technik, bei der ein Kartenspiel scheinbar gemischt wird, während die Reihenfolge teilweise oder vollständig erhalten bleibt.

## Beschreibung

Der Zuschauer sieht einen überzeugenden Mischvorgang, doch der Magier behält die Kontrolle über bestimmte Karten oder das gesamte Deck. Falschmischen ist ein essenzielles Werkzeug, um Skepsis zu zerstreuen — wenn das Deck „gerade erst gemischt wurde", glaubt der Zuschauer nicht an vorbereitete Reihenfolgen.

## Varianten

### False Overhand Shuffle
Der Overhand Shuffle ist optisch simpel, kann aber so ausgeführt werden, dass eine oder mehrere Karten an ihrer Position bleiben — z.B. die oberste Karte oben oder eine Karte von unten nach oben gebracht wird. Gut für Anfänger lernbar.

### False Riffle Shuffle
Beim Riffle Shuffle werden zwei Hälften ineinandergeflochten. In der falschen Variante wird das Zusammenführen so gesteuert, dass bestimmte Karten oder die gesamte Reihenfolge erhalten bleibt.

### Zarrow Shuffle
Ein besonders täuschender False Riffle Shuffle, benannt nach Herb Zarrow. Das gesamte Deck bleibt in seiner Originalreihenfolge — obwohl der Shuffle überzeugend echt wirkt. Gilt als eine der besten False Shuffles überhaupt.

### Overhand Run
Eine schnelle Variante des Overhand Shuffle, bei der einzelne Karten kontrolliert werden können. Gut geeignet, um eine Karte gezielt von unten nach oben oder in eine bestimmte Position zu bringen.

## Verwendung in Tricks

- Jeder Trick, bei dem eine „gewählte Karte" nach dem Mischen kontrolliert werden muss
- Memorized-Deck-Routinen (das Deck behält seine eingestudiete Reihenfolge)
- Stack-Arbeit (Si Stebbins, Mnemonica)
- Als psychologisches Mittel: „Ich mische — siehst du? Alles fair."
