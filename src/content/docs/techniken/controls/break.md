Ein Break (auch: Pinky Break) ist ein kleiner, unsichtbarer Spalt zwischen zwei Kartenpaketen im Deck, der mit einem Finger gehalten wird. Er markiert eine Position und ist Voraussetzung für viele andere Techniken.

## Beschreibung

Ein Break ist keine eigenständige Zaubertechnik, sondern ein **Hilfsmittel** — eine vorbereitende Handposition. Der Magier hält einen winzigen Spalt im Deck, meist mit dem kleinen Finger (Pinky Break), um eine bestimmte Kartenposition zu markieren.

Aus einem Break heraus kann man direkt in andere Techniken übergehen:
- [Double Lift](/techniken/double-lift) — ein Break unter den obersten zwei Karten
- [Forces](/techniken/forces/) — ein Break markiert die Position der Force-Karte
- [Pass](/techniken/controls/pass) — der Break zeigt die Teilungsstelle

## Ausführung

1. Eine Karte wird ins Deck gesteckt oder eine Position wird markiert
2. Der kleine Finger schiebt sich leicht unter das obere Paket
3. Ein minimaler Spalt entsteht — von vorne nicht sichtbar
4. Die Hand hält das Deck natürlich, der Spalt bleibt erhalten

Der Break muss so klein sein, dass er von keinem Winkel aus sichtbar ist.

## Wichtige Varianten

- **Pinky Break** — der häufigste Break, mit dem kleinen Finger gehalten
- **Thumb Break** — mit dem Daumen gehalten, nützlich bei bestimmten Handpositionen
- **Jogged Card** — eine leicht versetzte Karte als sichtbarer (aber unauffälliger) Marker im Deck
- **Push-Through Break** — die Karte wird hindurchgestoßen und das überstehende Ende wird zum Erzeugen des Breaks genutzt (siehe unten)

## Push-Through Break (Methode nach Matthias Berger)

Eine besonders elegante Methode, einen Break zu erzeugen, ohne aktiv einen Finger in das Deck zu schieben:

1. Die gewählte Karte wird mittig ins Deck gesteckt und etwa **2 Finger breit** vorne herausschauen gelassen
2. Das Deck wird mit einer Hand locker an den **langen Seiten** gehalten — nicht zu fest
3. Mit der anderen Hand wird die herausschauende Karte **kräftig ins Deck hineingestoßen**, sodass sie auf der gegenüberliegenden Seite ein kleines Stück herausschaut
4. Die Hand wechselt: Das Deck wird von oben gegriffen
5. Der **Daumen** biegt das leicht herausschauende Ende der Karte minimal nach unten
6. Dadurch entsteht auf der vom Zuschauer abgewandten Seite ein sauberer Break

**Warum diese Methode funktioniert:** Die Stoßbewegung ist für den Zuschauer eine natürliche, sichtbare Handlung ("ich stecke die Karte ins Deck"). Das Überstehen auf der Rückseite ist minimal und nur vom Magier sichtbar. Der Daumen nutzt dieses Ende als diskreten Hebelpunkt.

## Verwendung in Tricks

Kaum ein Kartentrick kommt ohne Break aus. Er ist die unsichtbare Verbindung zwischen dem Moment, wo eine Karte zurückgesteckt wird, und dem Moment, wo der Magier sie kontrolliert.
