Der Elmsley Count (auch: *Ghost Count*) ist eine Falschzählung für ein Päckchen von vier Karten. Beim Abzählen wird eine bestimmte Karte zweimal gezeigt — eine andere Karte bleibt dabei komplett verborgen. Für das Publikum sieht es aus, als würden vier Karten einzeln gezählt.

## Geschichte

Entwickelt von **Alex Elmsley** (1929–2006), einem britischen Mathematiker und Zauberkünstler. Er veröffentlichte die Technik in den 1950er-Jahren in *The Gen* (einem britischen Zaubermagazin). Der Count wurde rasch zur Standardtechnik für Packet Tricks und ist bis heute einer der meistgenutzten Falschzählungen.

## Beschreibung

Klassisches Beispiel: Du hast vier Karten — drei Asse und eine Blanko-Karte (oder eine andere Karte, die du verbergen willst). Mit dem Elmsley Count zeigst du scheinbar vier Asse.

Die vier Karten liegen als Päckchen face-down in der nicht-dominanten Hand. Beim Zählen werden sie einzeln in die andere Hand übernommen — scheinbar fair, eine nach der anderen. Tatsächlich wird beim zweiten Zug die verborgene Karte unter das Päckchen geschoben: Sie wird nie gezeigt. Stattdessen erscheint ein Ass ein zweites Mal.

**Beispiel mit 4 Karten (von oben nach unten: A♠, Joker, A♥, A♦):**

| Zählschritt | Das Publikum sieht | Was wirklich passiert |
|---|---|---|
| „Eins" | A♠ | Oberste Karte wird normal abgenommen |
| „Zwei" | A♥ | Joker wird versteckt, A♥ wird stattdessen gezeigt |
| „Drei" | A♠ (nochmals) | A♠ erscheint ein zweites Mal |
| „Vier" | A♦ | Letzte Karte wird normal abgenommen |

Das Publikum sieht vier Asse — der Joker bleibt unsichtbar.

## Ausführung

Ausgangspunkt: Vier Karten face-down im [Mechanic's Grip](/techniken/griffe/mechanics-grip) der linken Hand (bei Rechtshändern).

1. Rechte Hand nimmt die oberste Karte ab und hält sie
2. Linke Hand schiebt die neue oberste Karte nach rechts — rechte Hand nimmt sie aber **nicht einzeln**, sondern schiebt sie unter das Päckchen in der linken Hand (verdeckte Bewegung)
3. Gleichzeitig gibt die rechte Hand ihre gehaltene Karte auf das Päckchen — so entsteht der Eindruck, dass „zwei" gezählt wurden
4. Dritte Karte wird normal abgenommen
5. Vierte Karte wird normal abgenommen

Die zweite Karte befindet sich jetzt unten im Päckchen — unsichtbar gezählt.

## Varianten

### Buckle Count
Ähnliches Prinzip, aber die versteckte Karte wird durch eine leichte Biegung (*buckle*) der untersten Karte zurückgehalten. Einfacher zu lernen, etwas weniger überzeugend von der Seite.

### Hamman Count
Variante von Bill Hamman — hält zwei Karten statt einer verborgen. Für Tricks mit 5+ Karten.

## Verwendung

- **Packet Tricks** — klassischer Einsatz: vier Asse zeigen, obwohl nur drei vorhanden sind (oder umgekehrt)
- **[Oil and Water](/tricks/karten/oil-and-water)** — Trennung der Farben scheinbar bestätigen
- **Kartenveränderungen** — eine Karte hat sich „verwandelt", Elmsley Count beweist es scheinbar
- Überall wo die genaue Anzahl oder Identität der Karten im Päckchen verschleiert werden muss
