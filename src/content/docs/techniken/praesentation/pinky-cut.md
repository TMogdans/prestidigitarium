Der Pinky Cut ist ein eleganter einhändiger Schnitt, bei dem der kleine Finger das Deck in zwei Hälften teilt und die untere Hälfte nach oben befördert — alles in einer fließenden Bewegung mit einer Hand.

## Beschreibung

Das Deck liegt im [Mechanic's Grip](/techniken/griffe/mechanics-grip). Der kleine Finger greift ein Päckchen von unten und hebt es über das obere Päckchen — der Schnitt ist abgeschlossen. Die Bewegung wirkt mühelos und ist ein klassisches Präsentationselement zwischen Techniken.

Der Pinky Cut hinterlässt (sofern kein Break gehalten wird) eine zufällige neue Kartenreihenfolge — er wird deshalb oft als **scheinbar unverfängliche Mischung** eingesetzt. Mit einem [Break](/techniken/controls/break) kombiniert, kann die Reihenfolge dabei erhalten bleiben.

## Ausführung

1. Deck im Mechanic's Grip der nicht-dominanten Hand
2. Kleiner Finger schiebt sich unter die untere Hälfte des Decks — an der kurzen Kante
3. Kleine Finger hebt die untere Hälfte leicht an
4. Daumen und Zeigefinger stabilisieren die obere Hälfte
5. Die untere Hälfte wird über die obere gebracht und dort abgelegt
6. Der Schnitt ist abgeschlossen — das Deck liegt wieder vollständig in der Hand

## Mit Break kombiniert

Wenn vor dem Schnitt ein [Break](/techniken/controls/break) gehalten wird (z.B. über eine kontrollierte Karte), kann der kleine Finger genau an dieser Stelle schneiden — die Zielkarte bleibt an der gewünschten Position.

## Verwendung

- **Präsentation** — eleganter Übergang zwischen zwei Techniken
- **Scheinmischung** — sieht zufällig aus, kann aber mit Break kontrolliert werden
- **Vorbereitung** — Deck in Position bringen ohne auffällige Bewegung
- Als Alternative zum [Charlier Cut](/techniken/praesentation/charlier-cut) wenn beide Hände frei sind

## Vergleich mit ähnlichen Schnitten

| | Pinky Cut | Charlier Cut |
|---|---|---|
| Hände | Eine | Eine |
| Finger führend | Kleiner Finger | Daumen + Zeigefinger |
| Schwierigkeit | Mittel | Mittel |
| Kontrollierbar | Ja (mit Break) | Ja (mit Break) |
| Wirkung | Kraftvoll | Elegant |
