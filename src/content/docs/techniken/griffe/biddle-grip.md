Der Biddle Grip ist ein Griff für die dominante Hand, bei dem das Deck von oben gehalten wird — Daumen unten, alle vier Finger oben. Unverzichtbar für viele Flourishes und bestimmte Sleights.

## Beschreibung

Das Deck wird von oben gegriffen: Der Daumen liegt an einer kurzen Kante unten, Zeige-, Mittel-, Ring- und kleiner Finger liegen oben auf dem Deck. Das Deck hängt gewissermaßen von den Fingern — eine komplett andere Mechanik als der [Mechanic's Grip](/techniken/griffe/mechanics-grip).

## Ausführung

1. Dominante Hand über das Deck führen
2. Daumen an der unteren kurzen Kante ansetzen
3. Zeige-, Mittel-, Ring- und kleiner Finger oben auf dem Deck
4. Das Deck wird von oben gehalten, Handfläche zeigt nach unten

## Verwendung

- **Swing Cut** — klassischer Flourish-Schnitt aus dem Biddle Grip
- **Dribble** — Karten fallen aus dem Biddle Grip nach unten
- **Übergabe** — Deck von einer Hand in die andere mit stilvollem Griff
- **Cardistry** — viele moderne Flourishes starten im Biddle Grip
- Als Ausgangspunkt für den [Charlier Cut](/techniken/praesentation/charlier-cut)
