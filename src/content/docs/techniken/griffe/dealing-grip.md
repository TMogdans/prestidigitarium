Der Dealing Grip ist die natürlichste Art, ein Deck zu halten — so wie man Karten beim normalen Kartenspiel austeilt. Er wirkt ungekünstelt und unauffällig.

## Beschreibung

Das Deck liegt flach in der Hand, die Finger liegen locker darunter, der Daumen oben. Im Vergleich zum [Mechanic's Grip](/techniken/griffe/mechanics-grip) liegt das Deck etwas flacher und die Finger sind weniger präzise positioniert. Die Hand wirkt vollkommen normal — wie jemand, der gerade Karten ausgeben möchte.

## Ausführung

1. Deck face-down flach in die nicht-dominante Hand legen
2. Die Finger liegen entspannt unter dem Deck
3. Der Daumen liegt locker oben auf
4. Das Deck liegt flach, nicht seitlich

## Unterschied zum Mechanic's Grip

| | Dealing Grip | Mechanic's Grip |
|---|---|---|
| Abdeckung | Gering | Hoch |
| Natürlichkeit | Sehr hoch | Mittel |
| Kontrolle | Geringer | Höher |
| Für Sleights | Eingeschränkt | Ideal |

## Verwendung

- **Austeilen** von Karten — die offensichtlichste Verwendung
- Als **neutrale Haltung** wenn keine Technik ausgeführt wird
- Für bestimmte [Forces](/techniken/forces/) die eine natürliche Präsentation erfordern
- Als **psychologisches Mittel** — sieht weniger „magisch" aus als der Mechanic's Grip
