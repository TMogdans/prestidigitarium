## Task 3: Extend content schema with custom frontmatter fields

**Files:**
- Modify: `src/content.config.ts`

**Step 1: Update content.config.ts**

```typescript
import { defineCollection, z } from 'astro:content';
import { docsLoader } from '@astrojs/starlight/loaders';
import { docsSchema } from '@astrojs/starlight/schema';

export const collections = {
  docs: defineCollection({
    loader: docsLoader(),
    schema: docsSchema({
      extend: z.object({
        tags: z.array(z.string()).optional(),
        effekt: z.string().optional(),
        schwierigkeit: z.enum(['leicht', 'mittel', 'schwer', 'experte']).optional(),
        requisiten: z.array(z.string()).optional(),
        typ: z.enum(['gimmick', 'normal', 'technik', 'prinzip']).optional(),
      }),
    }),
  }),
};
```

Misdirection (deutsch: Irreführung, Ablenkung) ist das fundamentalste Prinzip der Zauberkunst. Es beschreibt die Kunst, die Aufmerksamkeit des Publikums gezielt zu lenken — weg vom Geheimnis, hin zum Effekt.

## Beschreibung

Misdirection ist keine einzelne Technik, sondern ein übergreifendes Prinzip. Der Magier nutzt natürliche menschliche Aufmerksamkeitsmuster aus, um das Geheimnis zu verbergen.

## Arten von Misdirection

### Physische Misdirection
Aufmerksamkeit wird durch Bewegung, Blickrichtung oder Gesten auf einen bestimmten Punkt gelenkt.

### Zeitliche Misdirection
Der geheime Move geschieht zeitlich versetzt — nicht im „magischen Moment", sondern davor oder danach, wenn das Publikum nicht aufpasst.

### Psychologische Misdirection
Durch Humor, Fragen oder Geschichten wird der Geist des Zuschauers beschäftigt, während die Hände arbeiten.

## Verwendung in Tricks

Misdirection wird in praktisch jedem Zaubertrick eingesetzt, besonders bei:
- Sleight-of-Hand-Tricks (Karten, Münzen)
- Bühnenillusionen (Ablenkung durch Assistenten)
- Mentalmagie (Ablenkung durch Patter)

## Quellen & Referenzen

- *Das große Buch vom Zaubern* — Bruce Smith

Palmieren (engl. Palming) ist die Technik, ein Objekt unsichtbar in der Hand zu verbergen, während die Hand natürlich und leer aussieht.

## Beschreibung

Beim Palmieren wird ein Objekt (Karte, Münze, Ball) in einer natürlichen Handposition verborgen gehalten. Der Zuschauer soll nicht merken, dass die Hand nicht leer ist.

## Varianten

### Classic Palm (Münzen)
Die Münze wird durch Muskelspannung im Handteller gehalten.

### Finger Palm
Das Objekt wird in den gekrümmten Fingern gehalten — natürlicher, aber weniger sicher.

### Back Palm (Karten)
Die Karte wird hinter der Hand verborgen, während die Handfläche leer gezeigt wird.

## Verwendung in Tricks

- Münztricks (Verschwinden, Wanderung)
- Bälle (Becherspiel, Multiplying Balls)
- Karten (Produktion, Verschwinden)

## Quellen & Referenzen

- *Das große Buch vom Zaubern* — Bruce Smith
